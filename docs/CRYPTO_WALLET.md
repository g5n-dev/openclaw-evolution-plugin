## 🔒 安全提示

1. **仔细核对地址**
   - 转账前请务必核对地址的前后几位字符
   - 不要只看中间部分

2. **小额测试**
   - 大额捐赠前建议先发送小额测试
   - 确认地址正确后再发送全额

3. **保留交易记录**
   - 保存交易哈希（TX Hash）
   - 截图转账记录
   - 发送邮件至 sponsors@openclaw-evolution.org 通知我们

4. **网络选择**
   - 确保在正确的网络上转账（主网 vs 测试网）
   - ERC-20 代币必须在 Ethereum 主网
